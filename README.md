# swimhub-legal

Legal documents and compliance pages for SwimHub mobile app.

## Documents

- **Privacy Policy** (`privacy-policy.md`) - Data collection and usage policy
- **Account Deletion** (`account-deletion.html`) - User account deletion request page

## Hosting

This repository is hosted via GitHub Pages:
- Privacy Policy: https://[username].github.io/swimhub-legal/privacy-policy.html
- Account Deletion: https://[username].github.io/swimhub-legal/account-deletion.html

## Setup

1. Enable GitHub Pages in repository settings
2. Set source to main branch
3. Update Google Form embed code in `account-deletion.html`
4. Update support email in `account-deletion.html`
